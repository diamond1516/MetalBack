## Getting Started

Setup project environment with [virtualenv](https://pypi.org/project/virtualenv/) and [pip](https://pypi.org/project/pip/).

```bash
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirementS/develop.txt


```
